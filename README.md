# 🌐 Kişisel Portföy Web Sitesi

Bu proje, kişisel portföyümü, deneyimlerimi, yeteneklerimi ve projelerimi sergilemek amacıyla geliştirdiğim modern bir web sitesidir.

## 🚀 Hakkımda

Portföy sitesi; ziyaretçilerin benim hakkımda bilgi edinmesini, çalışmalarımı incelemesini ve benimle iletişime geçmesini kolaylaştırmak için tasarlanmıştır.

## ✨ Özellikler

* Modern ve responsive tasarım
* Hakkımda bölümü
* Deneyim ve eğitim bilgileri
* Proje vitrin alanı
* Yetenekler ve teknolojiler bölümü
* İletişim sayfası
* Hızlı ve kullanıcı dostu arayüz

## 🛠️ Kullanılan Teknolojiler

* HTML5
* CSS3
* JavaScript
* Flask
* Python

## 🎯 Amaç

Bu projenin amacı; profesyonel geçmişimi, teknik becerilerimi ve geliştirdiğim projeleri tek bir platform üzerinde sergileyerek ziyaretçilere kapsamlı bir portföy deneyimi sunmaktır.

## 📬 İletişim

Sorularınız veya iş birliği teklifleri için web sitesi üzerindeki iletişim formunu kullanabilirsiniz.
